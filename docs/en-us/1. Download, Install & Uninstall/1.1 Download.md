# 1.1 Download

## Versions

The TPA is available in 2 versions：

### Light
- No Macro functions
- Simple structure with high stability
- Affects similar to Vanilla /tp command 
- Teleport flow: requester → directly teleports to the target
**Best for:**
- Small private server
- Local Area Network play
---

### Extra
- More game mechanics with macro functions
- Complete request workflow provided
- Supporting multiple interactions like cancel, accepts and denies

**Best for:**
- Small to medium community servers
- Servers that need a more polished teleport experience

---

> For large public servers, we generally recommend using plugins instead of the datapack.
> Larger servers usually benefit from more mature and better-optimized plugin or mod solutions.
> If you need help choosing or setting one up, feel free to contact us.


If you are using a version below 1.13, please refer to: [Command block solution for pre-1.13 and Bedrock Edition](docs/en-us/1.%20Download, Install & Uninstall/1.6%20Command Block Solution for pre-1.13 & BE.md)

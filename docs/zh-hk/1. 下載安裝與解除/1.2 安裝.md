# 1.2 安裝

安裝方法基本上同其他 Datapack 一樣。

## 喺雲端伺服器上

1. 打開你所託管伺服器嘅控制台。
2. 搵到並撳下標示為 *File Manager* 嘅按鈕。
3. 之後嘅步驟同本地伺服器安裝方法相同。

## 喺本地伺服器上

1. 搵到包含伺服器 JAR 檔案嘅伺服器資料夾。
2. 如果有 `world` 資料夾，請打開 `./world/datapacks/` 目錄。
3. 如果冇，或者完成第 4 步之後仍然無效，請檢查 `server.properties -> level-name`，搵出正確嘅存檔資料夾。
4. 將 Datapack 檔案放入 `datapacks` 資料夾內。
5. 進入伺服器，執行 `/reload`，之後執行 `/datapack list`，確認 Datapack 已成功識別。

## 喺單人世界中

1. 離開目前世界。
2. 喺世界選擇畫面揀選目標世界。
3. 撳左下角嘅 `編輯` 按鈕。
4. 撳 `打開世界存檔文件夾` 按鈕，如果有呢個按鈕，直接跳到第 8 步。
5. 返回世界選擇畫面，記低遊戲內標題下方嘅灰色文字。
6. 返回主選單，依次撳選 `選項…… -> 資源包…… -> 打開包文件夾`
7. 然後切換到上一層資料夾（通常係 `.minecraft`），進入 `saves` 資料夾，再進入之前記低嘅灰色文字所對應嘅資料夾（路徑類似 `../saves/(你的世界)/datapacks`）。
8. 進入 `datapacks` 資料夾，將 Datapack 檔案放入其中。
9. 進入世界，執行 `/datapack list` 確認 Datapack 已成功識別。

## 喺新建立嘅世界中

1. 撳 `更多 -> 數據包 -> 打開包文件夾`。
2. 將 Datapack 檔案放入其中。
3. 建立世界。
4. 執行 `/datapack list` 確認 Datapack 已成功識別。

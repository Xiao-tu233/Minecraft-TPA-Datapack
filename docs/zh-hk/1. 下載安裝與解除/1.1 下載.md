# 1.1 下載

## 版本區分

TPA Datapack 分為兩個版本：

### 輕量版（Lite）

* 唔使用巨集函數（Macro Function）
* 結構簡單，穩定性高
* 功能接近原版 `/tp + 玩家`
* 傳送流程：請求者 → 直接傳送到目標

**適合：**

* 玩家數量較少嘅伺服器
* 區域網連線（內網穿透／Radmin LAN／同一區域網）

---

### 完整版（Extra）

* 使用巨集函數及更多遊戲機制
* 提供完整嘅請求流程
* 支援接受／拒絕／取消等互動功能

**適合：**

* 小型至中型伺服器
* 需要更完整傳送體驗嘅場景

---

> 對於大型伺服器
> 從服主嘅角度出發，我哋並唔建議使用 TPA Datapack。
> 呢類伺服器通常會有更成熟、效能更佳嘅 Plugin 或 Mod 方案。
> 不過如果你喺選擇或部署 Plugin 時遇到困難，歡迎向我哋查詢。

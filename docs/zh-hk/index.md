# TPA Datapack 用戶手冊

# 簡介
TPA Datapack 係一個 Minecraft 數據包，容許玩家喺冇 OP 權限嘅情況下互相傳送。相比起其他提供同類功能嘅 Mod／Plugin，呢個 Datapack 支援語言功能，而且設定可以輕鬆調整，幾乎所有功能都可以調整。我哋亦樂意聽取用戶提出嘅每一項建議，希望你會鍾意呢個 Datapack。

# 閱讀須知
每個頁面前都會標示其支援嘅版本，請留意分別。

文中所有伺服器設定均透過 `/function tpa:options` 指令修改，所以進行呢啲更改需要 OP 權限。

更改設定時，撳掣後唔會出現任何提示訊息，屬正常現象。設定有冇成功套用，請以相關功能係咪生效／停用作準。

部分設定需要填寫指定數值。撳下「設定」之後，請喺提示指令嘅最後補上需要填寫嘅數值，再按 Enter 套用設定。

文中所有內容均以最新版本為準，請及時更新至最新版。

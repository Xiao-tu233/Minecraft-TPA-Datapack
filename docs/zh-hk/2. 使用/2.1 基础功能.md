# 2.1 基本功能

使用 `/trigger tpa`，或者喺進入遊戲後撳左下角嘅 `[撳我開啟 TPA 選單]` 來開啟 TPA 選單。

透過 `/trigger tpa` 指令可以開啟 TPA 選單。無論任何情況，都請記住呢條唯一需要記住嘅指令。

當一個請求被送出後，請求者可以喺對方接受之前取消請求；未開啟自動接受功能嘅被請求者，則可以喺請求逾時前選擇接受或拒絕。

接受請求可以透過撳聊天欄中嘅「接受」按鈕，或者手動輸入 `/trigger tpaccept` 來接受請求。

當請求逾時後，請求者嘅取消操作，以及被請求者嘅接受／拒絕操作都會失效，並顯示相關提示。

逾時時間可以喺設定介面中調整，單位為遊戲刻（Game Tick），預設值為 6000，即喺運作流暢嘅情況下約為 30 分鐘。透過設定為 `0` 或 `-1`，可以分別達到禁止傳送及永不逾時嘅效果。

需要注意嘅係，請求係同發出請求嘅玩家綁定。換句話講，每位玩家同一時間只可以持有一個請求（無論係傳送請求定係傳送到此處請求）。當新請求被送出時，舊請求會自動取消。

輕量版並冇傳送請求、接受、取消、逾時等概念。喺輕量版中，玩家發出傳送請求後，會即時被傳送到目標玩家所在位置。

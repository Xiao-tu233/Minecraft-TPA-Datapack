版本兼容/支持：1.20.2+ v2.0.3+完整版

该功能更多地被称为*home*即"家" 但事实上在许多语境中 这个此并不能满足表达的需要和通畅 且不符合汉语多音节词作为术语的韵律规律和语言习惯 故我将使用"传送点"作为*home*的翻译

使用/trigger tpa.home set -1 或在TPA菜单中点击"[传送点菜单]"来打开传送点菜单 根据设置菜单中设置的允许传送点个数（默认3个） 会向玩家开放对应的槽位 传送点可以被设置在任意维度的整数坐标 并且不与数据包ID绑定而与玩家名绑定 故无论玩家上下线 传送点将被保留 

如果对应位置的传送点被建立 传送点# 字样将存在悬停事件 玩家可以将鼠标悬停来检视传送点储存的坐标 及通过颜色来观察其维度 传送按钮会将你传送到传送点 移除会将传送点移除 而设置传送点将会把现在的坐标覆盖掉原来所储存的传送点槽位

使用/trigger home 可以快捷传送到传送点#1

使用/trigger home set ID 可以快捷传送到传送点#ID

# 2.2 传送功能进阶
## 自动接受
在完整版中，玩家可以在TPA菜单中，或者使用命令`/trigger tpaccept set 2`来开启自动接受。
开启后，可以同样通过菜单，或者使用命令`/trigger tpaccept set 3`来关闭。
开启自动接受会自动接受任何人向自己发送的收到的传送请求，但传送此处请求仍然需要手动处理。

## 传送此处
传送此处（tpahere）是一种传送方式，由请求者向目标玩家发送请求，使其传送至请求者所在位置。
发送传送此处请求，需要调出传送此处菜单，它长得和传送菜单很像，但上方的文字提示有所不同。
在传送此处菜单中，和正常的传送一样，直接点击名称即可发送传送请求。
传送此处的传送目标由记分项`tpa.tpahere`控制。



## 历史
| 版本 | 内容 |
| :-- | :-- |
| 1.7以前 | 加入自动接受 |
| 1.9 | 加入传送此处请求 |
| 2.0 | 自动接收改由由记分项tpa.tpaccept_toggle(tpa.tpaccept_tgl)控制 |
| 2.0.6 | 传送此处请求的目标现在不储存在记分项`tpa.tp_here`中，而是转而存在`tpa.tp_to`原来的记分项用来作为布尔值储存当前发送的请求是传送请求还是传送此处请求 |

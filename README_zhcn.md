## 优点

- 它正在服务器上运行。它根本不需要修改服务器，这会让一切保持原样. 
- 多平台/服务器兼容性。支持Linux/Windows上的Vanilla, Paper, BungeeCord等. 

## 咋工作的?

凭借1.13以来内置的数据包，我们通过编写一个包含函数文件的数据包来实现该功能.

## 优势

* 所有选项都可以通过 `/function tpa:options` 命令进行编辑。无需担心因默认设置不当导致的体验问题。
* 安装方式非常简单: 使用数据包。数据包是最容易安装到 Minecraft 中的内容类型，无需任何扩展或加载器。
* 我们也会积极确保与其他数据包/模组/插件/资源包的兼容性。因此，即使遇到可能不是由 TPA 数据包引起的错误，您仍然可以提供完整的上下文并告诉我们如何复现问题。
* 我们提供两个版本供您选择: 完整版 版本和 轻量版 版本。完整版 版本包含我们开发的所有功能，而 轻量版 版本仅包含基本功能，以满足不同需求。

## 工作原理

依赖于自 Minecraft 1.13 起的原版功能数据包（Datapack）实现

## 快速开始
    
1. 确认你的 Minecraft 版本
2. 在 Releases 中下载对应版本的数据包（完整版 / 轻量版）
3. 将 zip 文件放入世界的 `datapacks` 文件夹
4. 进入世界或服务器后执行 `/reload`
5. 执行 `/datapack list` 确认已加载
6. 输入 `/trigger tpa`. 若出现菜单，即表示数据包已成功加载

不同环境（服务器 / 本地已有存档 / 新建存档 / 云服面板服）的详细安装步骤，请参阅：
-> [详细安装步骤请参阅安装文档](docs/zh-cn/1.%20下载安装卸载/1.2%20安装.md)

### 如何选择下载版本

请使用下表确定您的 Minecraft 版本对应的数据包文件后缀: 
**Minecraft 版本与数据包文件名后缀对应表**

| 遊戲版本 | 完整版 | 轻量版 |
| :-- | --: | --: |
| 1.13 - 1.14.4 | +mc1.13 | light+mc1.13 |
| 1.15 - 1.20.1 | +mc1.15 | light+mc1.13 |
| 1.20.2 - 1.20.6 | +mc1.20.2 | light+mc1.13 |
| 1.21 - 1.21.4 | +mc1.20.2 | light+mc1.21 |
| 1.21.5 - | +mc1.20.2 | light+mc1.21.5 |

轻量版 最新版本: v2.0.6
完整版 最新版本: v2.0.6

如果你不确定该选哪个版本，请直接记住这条规则：

- 想要 请求和接受 / Back / Home / Warp / 多语言 → **完整版**
- 只需要基础的玩家互相传送 → **轻量版**

示例: 
如果我的 Minecraft 版本是 1.20.1，并且我想使用 完整版 版本，我会选择名为 `TPA_datapack-v2.0.6+mc1.15.zip` 的文件。

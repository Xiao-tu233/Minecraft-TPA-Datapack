## 優勢

-它正在伺服器上運行。 它根本不需要修改伺服器，這會讓一切保持原樣.

-多平臺/伺服器相容性。 支持Linux/Windows上的Vanilla，Paper，BungeeCord等.

## 如何工作? 

憑藉1.13以來内置的資料包，我們通過編寫一個包含函數檔案的數据包來實現該功能.

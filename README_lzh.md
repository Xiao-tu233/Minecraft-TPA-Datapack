## 錄囊之長處  

-    
一切設定，皆可行 `/function tpa:options` 指令編輯。毋須慮預設不善而致體驗不佳。  

-    
安裝極易：直置錄囊於礦藝中。毋需他載入器或附屬模組。  

-    
吾輩亦勤保兼容於諸他錄囊、模組、插件及資源包。即或遇錯誤非源於 TPA 錄囊，亦可具上下文告知復現之法。  

-    
提供兩版：**全備版**與**簡易版**。全備版載諸般功能，簡易版僅存基本傳送與選單，以應不同需。  

## 運行之法  

自礦藝 1.13 起，皆通原版介面而行。  

## 如何擇錄囊版本  

依下表辨貴版礦藝，取對應之錄囊檔名：  

| 遊戲版本 | 全備版 | 簡易版 |
| :-- | --: | --: |
| 1.13 - 1.14.4 | +mc1.13 | light+mc1.13 |
| 1.15 - 1.20.1 | +mc1.15 | light+mc1.13 |
| 1.20.2 - 1.20.6 | +mc1.20.2 | light+mc1.13 |
| 1.21 - 1.21.4 | +mc1.20.2 | light+mc1.21 |
| 1.21.5 - | +mc1.20.2 | light+mc1.21.5 |

簡易版最新：v2.0.6  全備版最新：v2.0.6  

1. 先察遊戲之版本。  
2. 擇全備版或簡易版，其差如左：  
   - 簡易版僅存基礎功能：傳送、選單。  
   - 全備版兼載諸般功能，如歸位、宅居、遨遊及多語言支持（未來或增更多）。  
3. 至 *Releases* 頁下載之。  
範例：若我礦藝版本 1.20.1，欲取全備版，則選 `TPA_datapack-v2.0.6+mc1.15.zip`。  

## 安裝方法  

大體與他錄囊同，若知之者可略讀此段。  

### 雲端伺服器  

1. 開托管伺服器之控制台。  
2. 點擊標 *File Manager* 按鈕。  
3. 其後步驟如本地伺服器安裝。  

### 本地伺服器  

1. 至伺服器資料夾，含 JAR 文件者。  
2. 若有 `world` 資料夾，則入 `./world/datapacks/`。  
3. 無之，或第四步不成，查 `server.properties -> level-name` 以尋正確存檔資料夾。  
4. 將錄囊放入 `datapacks` 資料夾。  
5. 入伺服器，行 `/reload`，復 `/datapack list` 以確認識別。  

### 單人世界  

1. 退當前世界。  
2. 選目標世界。  
3. 點左下 `編輯`。  
4. 點 `開啟世界資料夾`，若有此按鈕，可略去第 8 步。  
5. 記遊戲內標題下灰字。  
6. 主選單入 `選項... -> 資源包... -> 開啟資源包資料夾...`  
7. 上層切換至 `.minecraft`，入 `saves`，再入前記灰字對應之資料夾。  
8. 置錄囊於 `datapacks` 中。  
9. 入世界，行 `/datapack list`，以確認識別。  

### 新建世界  

1. 點 `更多 -> 資料包 -> 開啟資源包資料夾`。  
2. 置錄囊於其中。  
3. 建立世界。  
4. 行 `/datapack list` 以確認識別。
